File/Folder Structure

The files are placed in organized folders so to make it easy for you to find files where you will need to do customizations or change other stuffs easily.

Below is the folders and details of what they are for

  • css: This folder contains all style files or stylesheets.
  • webfonts: This folder contains all the Font Awesome used for in the Template.
  • img: This folder contains all the images
  • js: This folder contains all the scripts

Installation

Follow the steps below to get started with your Site Template:

  1. Open the Package/HTML Folder to find all the Templates Files
  2. You will need to Upload these files to your Web Server using FTP in order to use it on your Website.
  3. Make sure you upload the required files/folders listed below:


    • HTML/css - Stylesheets Folder
    • HTML/img - Images Folder
    • HTML/js - Javacripts Folder
    • HTML/webfonts - Font Awesome Icon
    • HTML/index.html - Index File/Homepage

HTML Grid

Using a single set of .col-lg-* grid classes, you can create a basic grid system that starts out stacked on mobile devices and tablet devices (the extra small to small range) before becoming horizontal on desktop (medium) devices. Contact Form

The contact form uses the PHP file for sending emails via contact form. By default, the email queries from the form goes to a dummy email address. To change the email to your email address which will receive the email queries from the contact form. Please follow the following steps.

  • Open the "contact.php" file which can be found in the main directory in any text editor.
  • In the second line, Change the "info@example.com" to your email.
